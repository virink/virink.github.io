    總結并記錄自己使用一些App的心得

    Base

    • iTerm 2
    • 很贊的一個終端，基本上是OSX的標配了
    • Alfred
    • 這也是神器，效率啟動器！！！
    • Google Chrome ＆ Firefox
    • 瀏覽器我就不多說了
    • Go2Shell
    • 在Finder下直接開啟終端的一款工具
    • FileZilla
    • FTP客戶端，挺好用的
    • TeamViewer
    • 用于实现远程支持、远程访问、远程管理、家庭办公及在线协作和会议功能的软件

    Reader

    • Dash
    • 離線文檔閱讀器
    • Reeder ＄
    • RSS訂閱客戶端
    • Pocket
    • 稍後閱讀的好工具

    Dev

    Editor & NoteBook

    • Sublime text 2
    • 強大的編輯器
    • Understand
    • 代碼閱讀神器
    • GitBook Editor
    • 沒事寫的東西咯
    • Leanote
    • 很好用的記事本，支持二次開發
    • MWeb
    • 試用中，有些地方不是很完美

    IDE

    • IntelliJ IDEA
    • Java IDE
    • PyCharm
    • Python IDE
    • PHPStorm
    • PHP IDE
    • CLion
    • C/C++ IDE
    • DataGrip
    • 數據庫客戶端
    • Android Studio
    • Android IDE

    Virtual

    • VirtualBox
    • 小型虛擬機，反正比VM小，湊合著用
    • Docker
    • 虛擬化容器
    • Kitematic
    • Docker UI 管理工具

    Other

    • Github Desktop
    • 版本控制，懶得用命令就用這個
    • AntSword
    • Webshell管理工具（類似中國菜刀）
    • OBS
    • 直播推送
    • TorBrowser
    • 。。。。。

    TrueWorld

    這幾個就不多說了，懂的自然懂
